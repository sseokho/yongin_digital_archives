기본 가이드

신규 페이지 생성시 DOCTYPE은 HTML5이며, 파일 인코딩은 UTF-8로 생성한다.
페이지 생성 및 유지보수 진행시 하단 코드를 참고하여 진행한다.
추가되는 css는 ux-admin.css 하단에 추가되는 해당 매뉴의 css 를 위치한다. 

                                    <!DOCTYPE html>
                                    <html lang="ko">
                                    <head>
                                        <meta charset="UTF-8">
                                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                        <link rel="stylesheet" href="../resource/css/ux-common.css">
                                        <script type="text/javascript" src="../resource/js/jquery-1.12.4.min.js"></script>
                                        <title>자율주행 데이터 공유센터 코딩가이드</title>
                                    </head>
                                    <body>
                                        <div class="wrap">
                                            <!-- header -->
                                            <header class="header">
                                                <div class="header__inner"></div>
                                            </header>
                                            <!-- container -->
                                            <section class="container">
                                                <div class="container__inner">
                                                    <div class="content">
                                                        <div class="content__inner">
                                                        </div>
                                                    </div>
                                                </div>
                                            </section>
                                            <!-- footer -->
                                            <footer></footer>
                                        </div>
                                    </body>
                                    </html>

CSS 정의

css 추가는 각 페이지 별로 따로 만들어 연결하되 공통 스타일은 common.css 쪽, 고정 style은 fix.css에 포함한다.
페이지 별 css는 sub__페이지명.css로 만들되 페이지별 head쪽 ux-common.css 아래에 선언한다.
plugIn css의 경우 css폴더 내 lib 폴더에 넣어서 연결한다. 

                                    <link rel="stylesheet" href="../resource/css/ux-common.css">
                                    <link rel="stylesheet" href="../resource/css/sub__status.css">

BUTTON

button은 <div class="btn-wrap>으로 감싸서 사용한다.
기본 버튼 사이즈는 width: 13.2rem; height:5.4rem; 이며,
width나 height 변경시 class 추가하여 사용함.
ex) btn--width-숫자 btn--height-숫자 

                                        <button type="button" class="btn--list">목록</button>
                                        <button type="button" class="btn--border-color-gray">수정하기</button>

BUTTON-RADIUS

아래와 같은 모양의 경우 button의 class에 btn--radius을 넣어준다.
.btn은 넣지 않는다. 

                                        <button type="button" class="btn--radius btn--color-aliceblue">CI 파일 다운로드</button>
